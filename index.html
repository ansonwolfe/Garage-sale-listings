Online Garage Sale

I got tired of reposting things on Craigslist and wanted to keep a list of stuff I could "thin" out so decided to build an online tool to help me with that. What I thought would have been a quick, one day thing turned out to be a week-long project.

Ironically, the main things I wanted to sell all got sold before I completed this. I'd like to add Twilio, phone-SMS to this for next batch of things I want to sell.

Site can be found here: Live on Heroku

Stack used:

  • Ruby on Rails
  • Paperclip gem
  • Amazon S3 web services (to host images)
  • Heroku
  • smidget of site tracking via MixPanel
